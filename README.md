## 📌 概要

記事タイトルや見出しを入力するだけで、**AI がデザインを判断してアイキャッチ画像を自動生成**するシステムです。

ブランドカラーや文字配置をテンプレート化しており、**記事ごとに統一感のある画像を量産**できます。

---

## ✨ 主な機能

- **テキストからアイキャッチを自動生成** — タイトル・サブタイトルを渡すだけで画像出力
- **16:9 サイズ対応** — ブログ・WordPress・note で使いやすいサイズに最適化
- **ブランドカラー統一** — テンプレートで配色・フォントを固定
- **バッチ処理対応** — 複数記事分をまとめて一括生成
